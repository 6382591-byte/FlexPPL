# FlexPPL V5

Changes:
- Gym busyness now uses an in-app four-button modal: Empty, Normal, Packed, or Cancel.
- Packed mode keeps the simple automatic reorder for crowded gyms.
- Exercise swaps now use a frictionless “Can't do this” modal with three one-tap temporary replacements and a Best replacement label.
- Weighted exercises include a plate calculator button beside the Weight field.
- Plate calculator opens in app, pre-fills the exercise weight, supports common bar weights, and shows plates to add per side.
