# FlexPPL Quick PWA

## Fastest deployment
1. Open your GitHub repository in a desktop browser.
2. Click **Add file → Upload files**.
3. Upload `index.html`, `manifest.webmanifest`, `sw.js`, and `icon.svg`.
4. Commit directly to `main`.
5. In Vercel, click **Add New → Project**, import the `FlexPPL` repository, and deploy.
6. Open the Vercel URL on Android Chrome.
7. Chrome menu → **Add to Home screen** / **Install app**.

All workout data stays on the device in localStorage. Use Settings → Export JSON for backups.
